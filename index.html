📁 フォルダ連携・分割保存と結合機能

OneDriveやSharePoint等の共有フォルダで複数人が衝突（競合コピーの発生）を避けながら同時に共同編集するための機能です。

🛠️ 使用ルール・書き方
  1. 分割の指定: 分割して別ファイルにしたい部分の直前に、<!-- split: ファイル名.md --> というコメント行を記述します。 
    <!-- split: 01_はじめに.md -->
# 1. はじめに
セクション1のテキストをここに書きます。
  2. 分割保存の実行: フォルダ連携した状態で「分割保存」ボタンをクリックすると、指定した各ファイル（例: 01_はじめに.md 等）と、結合用のインデックスファイル（元のファイル名）に自動分割されて保存されます。
  3. 結合読み込み（インデックスファイル）: 目次ファイルに自動生成される <!-- include: ファイル名.md --> の指示に基づき、ファイルを開いた際に同一フォルダ内の分割ファイルを自動結合して1つの連続したドキュメントとして画面に読み込みます。

GitHub リポジトリ連携

GitHub のリポジトリと直接連携し、ファイルを読み書きできます。
トークンはブラウザ内（LocalStorage）にのみ安全に保存されます。

ローカルドキュメント連携

ローカルの指定フォルダ（例: document フォルダ）と直接連携し、ファイルをシームレスに読み書きできます。
フォルダを選択すると、ファイルブラウザで内容を一覧選択・保存できます。
