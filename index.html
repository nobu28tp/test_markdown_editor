GitHub リポジトリ連携

GitHub のリポジトリと直接連携し、ファイルを読み書きできます。
トークンはブラウザ内（LocalStorage）にのみ安全に保存されます。

ローカルドキュメント連携

ローカルの指定フォルダ（例: document フォルダ）と直接連携し、ファイルをシームレスに読み書きできます。
フォルダを選択すると、ファイルブラウザで内容を一覧選択・保存できます。
